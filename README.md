# 39Wrt Support

此仓库作为 39Wrt 问题反馈仓库使用。

## 提交 Issue 前请注意

在反馈问题前，建议先进行以下操作：

- 尝试恢复出厂设置
- 使用浏览器隐身模式（无痕模式）
- 在清洁配置环境下重新测试问题

部分问题可能由于：

- 浏览器缓存
- 旧配置残留
- 插件冲突
- 升级后配置不兼容

导致出现异常现象。

如果在清洁环境下问题依旧存在，请按照 Issue 模板进行反馈。

---

## Issue 反馈建议

为了更快定位问题，请尽可能提供以下信息：

- 平台类型（Qualcomm / MediaTek / Rockchip / X86）
- 设备型号
- 固件版本
- 内核版本
- 问题详细描述
- 完整日志

日志越完整，越容易定位问题。

仅提供：

> “不能用了”  
> “炸了”  
> “看下”

通常无法有效分析问题。

---

## 日志建议

建议提供：

```bash
logread
dmesg
```

以及：

- 编译报错日志
- 崩溃日志
- 网络相关日志
- 无线相关日志

如日志较长，可使用代码块格式提交。

---

## 说明

本仓库仅用于问题反馈与跟踪。

请勿提交：

- 无关内容
- 重复 Issue
- 与第三方插件相关且无法复现的问题

感谢支持 39Wrt。
